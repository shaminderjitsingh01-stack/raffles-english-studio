2. Technology Stack

We're using a cutting-edge, performance-focused technology stack that ensures speed, security, and a premium user experience.

Component Technology Purpose
Frontend Framework Next.js 16 (React 19) Fast, SEO-friendly pages
Styling Tailwind CSS 4 Modern, responsive design
Animations CSS Animations + IntersectionObserver Smooth scroll-triggered effects
Language TypeScript Type-safe, reliable code
CMS Sanity CMS Client-managed blog content
Deployment Vercel Edge-optimised global hosting
Typography Playfair Display + Inter Premium serif + clean sans-serif

Why This Stack?

Performance: Next.js 16 with React 19 delivers blazing-fast load times through automatic code splitting and edge-optimised rendering. No WordPress bloat, no page builder overhead.

  • ✓ No WordPress vulnerabilities — Static and server-rendered pages can't be hacked
  • ✓ Zero database exposure — No SQL injection risks
  • ✓ Global CDN — Vercel serves from 100+ edge locations worldwide
  • ✓ Instant deployments — Push code, site updates in seconds
  • ✓ Self-managed blog — Sanity CMS lets the client publish articles without developer help
  • ✓ Future-proof — React/Next.js is the industry standard for 2026
